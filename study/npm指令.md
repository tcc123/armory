1. Sudo npm install npm -g
2. Npm help
3. Npm -v  查看版本
4. Npm init 初始化一个配置文件package.json
5. Npm init -y  快速生成配置文件
6. Npm search
7. Npm install <包名称>     -g 全局依赖  -s 项目依赖   -D 开发依赖
8. \-  ^  *最新包  @指定包
9. Npm run 运行指定脚本命令
10. Npm publish 发布包
11. Npm install -g yarn
12. Yarn init  
13. Yarn add <包名称>  下载包
14. Nrm use <源名称>  切换源
15.  Live-server 启动HTTP服务
16. Treer 生成文件目录树   
17. Npm uni 删包
